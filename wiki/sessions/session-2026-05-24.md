## Rules for Future Agents

### Pi SDK v2 Critical Rules
- **NO `Pi.init()`** — Pi SDK v2 does not have an init method
- **Auth format**: `Pi.authenticate({ scope: [...], onIncompletePaymentFound: fn })` NOT `Pi.authenticate(array, callback)`
- **Endpoint**: `/api/auth/pi` for Pi SDK auth, `/api/auth/connect` for demo wallets
- **Never revert auth endpoint** without checking which version of the SDK is in use

### Architecture Rules
- **KDB (this wiki)** is for agents and team to track progress — NOT connected to axiomid-project code
- **Don't mix repos**: KDB fixes go here, axiomid fixes go in axiomid-project. Separate Vercel projects.
- **Don't add `Pi.init()`** — it doesn't exist in SDK v2
- **Don't delete `/api/auth/pi`** — it's used by `wallet-context.tsx` for Pi SDK auth
- **Copy-local** for aix-format code, not npm link or monorepo
